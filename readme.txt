This Plugin enables the Integration of the nexxPLAY Media Player into WordPress

== Description ==

Use this Plugin to easily integrate the nexxPLAY Embed Links into your WordPress Pages.
If you authorize the Plugin in the Settings Panel, you can even browse all the Media items, you have stored in nexxOMNIA to embed them without any Code into your Projects.
3Q nexx or your nexxOMNIA Admin can help you connecting the Plugin successfully.


== Frequently Asked Questions ==

= Can anyone use this Plugin? =

You need to have a valid nexxOMNIA Account for using this Plugin.
You cannot register directly from the Plugin Page.
You can contact us anytime on support.nexx@3q.video.


= What kind of items can I embed with the Plugin? =

You can embed Videos, LiveStreams, Scenes, Audio, Radio, Playlists, Audio Albums, Collections and Sets.
Since Version 2.3, the Plugin also supports Embedding Widgets.

= Is the Classic Editor supported =

This Plugin only works with the Block Editor. The Classic Editor is not supported in any Way.

= Can I use my own Embed URL? =

Since Version 2.3, the Plugin will respect a custom Embed URL.
